Ryan Goldstein: Ironclads

A game about a historic naval battle

Written In
While I originally planned to use Java, the project was created with Game Maker Studio.
Why does it exist?
This project was actually created for a history assignment, and therefore starts with a bit of historical exposition.
How does it work?
The player controls the CSS Merrimack to fight the USS Monitor in a naval battle. They have the options to accelerate, deccelerate, bank to port, bank to staboard, and firing port or starboard guns.
