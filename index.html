Systems Applications / Compilers

AST
A small project that converts an Abstract Syntax Tree into C source code

AST

An Abstract Syntax Tree is an intermediate representation between source code and the compiler's output. Each node is an operator or value, and each leaf is a value. This project was my first step towards writing compilers that targeted C, as it was intend to turn a data structure into executable C code. I didn't wrap every feature of the C language with an AST feature, but enough that I learned critical parts of writing compiler infrastructure.
Fwum
A work-in-progress programming language with a compiler

Fwum

Fwum was created out of my frustration with other programming languages. I found garbage collection unsatisfactory for game development, and reference counting just as bad, leaving just a few languages. Mainly, the candidates were C and C++, which are the too hot and too cold of a Goldilocks metaphor. C wasn't quite featured enough for me, but C++ had too much complexity. I created a grammar, a parser for the grammar, a semantic analyzer, a basic linker, and an output to C as a compile target for the language.
IslandScript
A Python-Inspired language that compiles to Javascript

Island Script

Island Script was my first compiler, intended to make the syntax of Javascript more like Python. At the time, I believed semantic indentation was obviously the correct path, and wanted to be able to use it for web development. The language is nearly semantically identical to Javascript, with only minor semantic tweaks.
Snakefile
A build system in Python that is similar to Makefile (pun intended)

Snakefile

When I first started creating complex C projects, I was using a simple Makefile that mapped each source file to an object file. However, changes to dependent headers would often break builds, which meant I was in need of a better solution. Autotools was too complex, and I didn't like that CMake required learning an entire new Domain-Specific Language, so I created Snakefile. It is intended to allow creating complex, turing-complete build files and to be easy to configure. However, incompatibility with Visual Studio project files eventually turned me away from the project.

FwumDeStego
A stegnonography program that can encode text in images, using Java

FwumDeStego

The program converts a string to a bitstring and maps it over the RGB values of the image. Even values indicate a 0 in the bitstring and odd values indicate a 1 in the bitstring. To read the value from the image, it follows the same order and uses modulo to read the string. The changes to the RGB values are invisible to the human eye but allow for the transmission of information.

Speech Generator

The program analyzes what words follow others most often, and attempts to reconstruct the same pattern and write in a similar fashion. It is a rather naive implementation that only takes a basic context, and most often generates nonsense. However, this is a basic demo of what a Markov chain is like.
