BasicStego

$ #Put some image in image.png $ cat "Hello world!" &gr; test.txt $ java -jar BasicStego.jar encode image.png text.txt $ java -jar BasicStego.jar decode image.png out.txt $ cat out.txt A program that demonstrates stegonography: the practice of hiding information in images. It creates a binary string out of a Java string and then sets the RGBA values to even or odd, depending if each bit is on or off. This can be reversed, retrieving the information.
