Written In
This project was written in C.
Why does it exist?
The project is a programming language that attempts to solve my frustrations with C and C++.
How does it work?
The project uses a custom parser and lexer, feeds it to a semantic analyzer, does a simple optimization pass, links to any required Fwum files, and outputs an executable.
