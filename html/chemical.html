Written In
This project was written in Python.
Why does it exist?
I was bored in chemistry class after finishing my work, and decided to create something fun.
How does it work?
The user inputs some string, and the program uses atomic symbols (H for Helium, Fe for Iron, etc.) to recreate the string. For example, Bacon would become BaCON (Barium Carbon Oxygen Nitrogen.)
